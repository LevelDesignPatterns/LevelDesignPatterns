Level Design Patterns

A level design pattern is a repeating configuration (of space and logic) which can be found again and again in video games. They are valuable and used repeatedly for a variety of reasons especially due to their predictable effect on the players behaviours.

There are other collections of level design patterns, e.g. some part of a thesis or for specific genre of game. Level design patterns from those other sources are also included here.

The data is a js/json database and hosted on github at https://github.com/LevelDesignPatterns/ get intouch if you want to help.

If you use this data in a system or for research please make sure you reference back to us. https://creativecommons.org/licenses/by/4.0/
